起步 →

简明优先

对以 markdown 为中心的项目结构，做最简化的配置，帮助你专注于创作。

Vue 驱动

享用 Vue + webpack 开发环境，在 markdown 中使用 Vue 组件，并通过 Vue 开发自定义主题。

性能高效

VuePress 将每个页面生成为预渲染的静态 HTML，每个页面加载之后，然后作为单页面应用程序(SPA)运行。
